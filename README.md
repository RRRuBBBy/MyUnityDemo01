# Unity工业设备可视化Demo

## 📌 项目简介

本项目基于 Unity 实现一个简易的工业设备可视化与交互系统，模拟实际生产场景中的设备监控与控制流程。
主要包含设备状态展示、控制逻辑以及简单的生产流程（传送带+加工件）模拟。

该Demo偏向工业应用场景中的三维可视化（类似数字孪生中的基础表现层），重点在于**设备状态与三维表现的联动**。

---

## 🎯 功能说明

### 1. 设备监控

* 显示设备名称、运行状态、温度等信息
* 支持点击设备进行选中与信息展示

### 2. 设备控制

* 支持设备启动 / 停止
* 控制逻辑与UI状态实时联动

### 3. 传送带流程模拟

* 使用长方体模拟传送带
* 加工件（Cube）自动生成
* 加工件沿传送带方向移动
* 离开工作区域自动销毁

### 4. 状态联动

* 设备运行时：开始生成加工件
* 设备停止时：停止生成
* 实现简单“控制 → 执行”的工业逻辑闭环

---

## 🧱 技术实现

* Unity（3D场景搭建、交互实现）
* C#
* TextMeshPro（UI显示）
* 简单定时器与状态机逻辑

---

## 🖥️ 场景说明

* **传送带（Conveyor）**：模拟生产线
* **加工件（Item）**：动态生成并移动
* **设备（Device）**：控制生产流程
* **UI面板**：展示设备状态与控制按钮

---

## 🔄 核心逻辑

```text
设备启动 → 生成加工件 → 加工件移动 → 离开区域销毁
设备停止 → 停止生成
```
